Project Description:

Our project is a toy train system that allows users to create custom tracks with note placement to produce music. 
As the train moves along the track, it plays the notes, helping users learn musical concepts in an interactive and engaging way. 
The system includes adjustable tempo and instrument settings to enhance creativity and exploration.
The design is accessible for users of all ages. 
For younger users, it introduces basic music concepts through play, while older users can use expansion packs, such as track junctions, to build more complex songs and experiment with composition.
